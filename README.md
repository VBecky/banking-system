# 🏦 Professional Banking System (Python GUI)

A modern desktop banking system built using **Python, Tkinter, and SQLite**.  
This project simulates real banking operations with a user-friendly graphical interface.

---

## 🚀 Features

- 🧾 Create new bank accounts  
- 🔐 Secure login system  
- 💰 Deposit money  
- 💸 Withdraw money  
- 🔁 Transfer money between accounts  
- 📊 Transaction history view  
- 🗃️ SQLite database storage  
- 🖥️ Modern GUI using Tkinter  

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- SQLite3 (Database)
- datetime module

---

## 📁 Project Structure

```bash
banking_system.py   # Main application file

banking_system.db   # Database file (auto-created)
```

---

## ▶️ How to Run

1. Install Python (3.8+ recommended)

2. Run the application:
```bash
python banking_system.py
```
or
```bash
python3 banking_system.py
```
---

## 🧠 How It Works

- Users create an account with name, password, and initial deposit  
- Each user gets a unique account number  
- Login system allows secure access  
- All transactions are stored in SQLite database  
- Dashboard shows balance and banking options  

---

## ⚠️ Important Notes

- This project is for **educational purposes only**
- Passwords are stored in plain text (not secure for real banking)
